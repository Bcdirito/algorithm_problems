## Median of Two Sorted Arrays
Given two strings s and t , write a function to determine if t is an anagram of s.


### Test Cases
s = "anagram", t = "nagaram" -> true

s = "rdaar", t = "arard" -> true

s = "eata", t = "data"  -> false
